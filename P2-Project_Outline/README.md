## Submission Instructions

### Overview

My project will be a payment automation app that allows those calculating payments manually to expedite the process by uploading data via a spreadsheet or .csv file. The web app will calculate payments automatically based on parameters set by the administrator, and employees will be able to log in to view their payments.

The idea for this project comes from having done payments manually through Excel for a previous job, which was time-consuming and error-prone.

### Features

1. User login with roles and access restrictions
2. CRUD (administrators, employees, payment data)
3. Ability to upload and extract data from .csv
4. Ability to search and sort info about users and payments
5. Mobile responsive front-end

### Technologies

C#
ASP.NET Core
EF Core
Bootstrap

### What I'll Have to Learn

1. Enabling user authentication with roles and access restrictions
2. Uploading and extracting data from .csv file or spreadsheet
3. Making a mobile responsive front-end with Bootstrap
